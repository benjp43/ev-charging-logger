# ⚡ EV Charging Logger (Web Version)

A Streamlit web app for logging and analysing home EV charging sessions.

**Features**
- Calculates cost, night/day split, and off‑peak percentage
- Compares home charging vs public charger rates
- Works on desktop and mobile
- CSV‑based data persistence

**Run locally**
```bash
pip install -r requirements.txt
streamlit run ev_logger_web.py
